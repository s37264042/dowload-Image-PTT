# dowload-Image-PTT
dowload Image in PTT (C# WinForm) For Windows
* [Demo Video](https://www.youtube.com/watch?v=hfTaLP3bc8U) 

## 特色
* 抓取 PTT 圖檔，可指定看板or網頁大量抓取圖檔

## 輸出格式
* 資料夾裡為圖檔

## 軟體介紹
